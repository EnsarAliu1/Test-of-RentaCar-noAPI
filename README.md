# Rent a Car - Frontend Showcase

Welcome to the **Rent a Car** project! This repository serves as a live demonstration of the frontend capabilities of our car rental platform.

## 🚀 Live Demo Experience

You can experience the application as if it were running with a real, live backend API. However, **this version is a simulated test environment built specifically for demonstration purposes.**

Because this application is hosted statically (e.g., on GitHub Pages), it doesn't currently connect to a real database or server. Instead, to provide you with a fully interactive experience:

- **Simulated API:** We handle all data manipulation directly in the browser using JavaScript arrays.
- **Local Persistence:** Your sessions, registrations, reservations, and admin panel changes are saved seamlessly to your browser's `localStorage`.
- This means you can add cars, register clients, manage messages, and make reservations exactly as you would in the production app, and your data will persist across page reloads on your device!

## 🌟 Features Available in This Demo

- **User Authentication:** Sign up, log in, and browse cars securely.
- **Car Catalog:** View our premium fleet, filter by fuel type or transmission, and check details.
- **Reservations:** Book your favorite vehicle with specific pickup and drop-off dates.
- **Admin Dashboard:** Log in as an administrator (view the `admins` data structure if needed) to manage vehicles, users, reservations, and messages dynamically without a real backend!

## 💻 Tech Stack

- HTML5 & CSS3
- Bootstrap 5
- Vanilla JavaScript (ES6+)
- LocalStorage Data Management

Enjoy exploring the application! If you refresh the page, your changes will still be there for as long as your browsing data is kept.

---

_Note: This is a frontend-centric test environment. Real production versions would connect securely to a backend server (e.g., using Node.js/json-server) to persist data globally._
